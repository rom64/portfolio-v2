Про меня

Меня зовут Роман

Меня зовут Роман. Я занимаюсь web-разработкой. Умею создавать адаптивные сайты, пользовательские интерфейсы, wordpress-темы. В работе использую HTML, CSS, SCSS, Gulp, Javascript, React, jQuery, PHP, Wordpress, Git

Web-разработка
Веб-разработка — процесс создания веб-сайта или веб-приложения. Основными этапами процесса являются веб-дизайн, вёрстка страниц, программирование на стороне клиента и сервера.
Wordpress-темы
Wordpress - это самая популярная и самая удобная площадка для веб-разработки. Изначально система создавалась как платформа для блогов, но с того времени прошло много лет, и за это время она смогла стать самым универсальным веб-решением.
React-приложения
React – это популярная библиотека языка программирования Javascript, которая применяется для создания пользовательских web-интерфейсов.
Адаптивный дизайн
Адаптивный дизайн – это конфигурация, в которой сервер отправляет одинаковый HTML-код на любые устройства, а размеры элементов страницы корректируются с помощью CSS.

Навыки

Что я использую в работе

HTML5

Именно он создает каркас сайта или приложения, а пятая версия позволит мне создавать более SEO-оптимизированную структуру продукта

CSS3

Благодаря стилям я создаю абсолютно любой внешний вид сайта или приложения. Все ограничивается только фантазией дизайнера!

Javascript

Этот язык позволяет мне оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое

React

React позволяет собирать сложный UI из маленьких изолированных кусочков кода, называемых «компонентами»

jQuery

Библиотека Jquery позволит ускорить разработку. Без необходимости интегрировать в проект я её не буду, но навык работы с ней пригодится

PHP

Это один из самых распространенных языков web-разработки. Я применяю его для создания сайтов и web-приложений любой сложности.

WORDPRESS

WordPress тема – набор css, js, php файлов, которые выводят информацию из базы данных в удобном виде, что позволяет конструировать практически любые проекты

Git

Git - система контроля версий. Она хранит информацию об изменениях в коде и обеспечивает удобство командной работы.

Gulp

Gulp — это таск-менеджер для автоматического выполнения часто используемых задач.

Webpack

Webpack — это сборщик модулей JavaScript с открытым исходным кодом. Он может преобразовывать внешние ресурсы, такие как HTML, CSS и изображения, если включены соответствующие загрузчики.

SCSS

SCSS — это метаязык на основе CSS, предназначенный для увеличения уровня абстракции CSS-кода и упрощения файлов каскадных таблиц стилей.

MySQL

MySQL — реляционная СУБД. Ползволяет выбрать таблицы типа MyISAM, поддерживающие полнотекстовый поиск и InnoDB, поддерживающие транзакции на уровне отдельных записей.
